> NodeJS project generator

## Getting started

- Install: `npm install -g generator-jb-node`
- Run: `yo jb-node`


## Commands

* `yo jb-node` shows a wizard for generating a new NodeJS project


## What do you get?

Scaffolds out a basic NodeJS project

```
.
├── .gitignore
└── package.json
```
